Status messages display as chat lines in the lower-left of the game screen when certain conditions apply.

They all have one of the six or so colors that the game uses and a player can replicate them in chat using this formula [c/XXXXXX:]. Write the text you want to color after the colon and enter a hexadecimal color code in the place of "XXXXXX". For more see Announcement Box.

NPCs[]

  • When an NPC (excluding the Old Man and Skeleton Merchant) spawns, the message "<Name of NPC> the <type of NPC> has arrived!" displays. The hex code is:  #327DFF.
  • On the Nintendo 3DS version, when the requirements for an NPC have been fulfilled, but there is no housing for the NPC, the message "<Name of NPC> would like to settle down." may display.
  • When an NPC dies, the message "<Name of NPC> the <type of NPC> was slain..." is displayed.
    • When the Angler, the Princess, or a town pet dies, the message "<Name of Angler/Pet/Princess> has left!" is displayed instead.
    • In both cases the hex code is:  #FF1919.
  • When the Traveling Merchant departs, the message "<Name of NPC> the Traveling Merchant has departed!" is displayed. The hex code is:  #327DFF.

Trivia[]

  • In the  Mobile version, the message "<Name of NPC> would like to settle down" used to display when the requirements for an NPC were fulfilled, but there was no housing for the NPC. This was removed in the 1.3.0.7 update.
  • Although the King Slime can spawn naturally, it does not have a message.
  • When a message appears (for something that is about to happen) and the world is exited, the event/boss summon will not occur after reopening the world after.
