Map viewers are 3rd-party programs that analyze saved maps and display their contents in one easily-viewable graphic.

Map viewers are useful for finding Floating Islands and other resources, as well as providing a total world view not otherwise attainable. Some programs also provide advanced tools for checking map variables, checking Chest contents, and even altering maps in various ways.

TerraMap lets you pan and zoom around in an interactive map and dynamically highlight items, items in chests, dungeons, NPCs, etc. It also supports headless, unattended world map image generation via command-line, complete with highlight capability.

An alternate version, TerraMap Online, is a web app version of TerraMap that lets you use all its features from an in-browser interface.

Terraria Worlds Viewer[]

Terraria Worlds Viewer

Type

Map Viewer

Author

Jupisoft

Platform

Latest version

Compatible with

Terraria 1.4.4.9 v4

Download

  • [1]
  • [2]

Terraria Worlds Viewer is a .NET Framework application capable of loading a Terraria world and display all the existing blocks and liquids inside. By default it uses a 1x zoom, where 1 pixel equals 1 block, but it also includes 2x, 3x and 4x zoom. It also shows the block coordinate where the mouse cursor is, as well as the ID of the block and the block name, and also the ID of the liquid and liquid name. It even shows the world seed. It can export the full world map as .PNG format, even supporting the current zoom level. Also allows to copy the world image and even the world seed. It's a very simple but quick and effective tool to get a full overview of any world or to find any chest, ore or treasure, since it draws the liquids with transparency, allowing to see if there's anything inside of them, as well as several options to filter or remove ores, liquids or walls. Finally the worlds will always be loaded as read-only, so they'll never be modified in any way.

Omnitool[]

Omnitool

Type

Map Generator/Viewer

Author

Berserker66

Platform

 

Latest version

18.5.1

Compatible with

Terraria 1.3.4.3

Download

  • Official Terraria Forums

An extensible, open source, tool collection that merges many conveniences of other tools into a GUI. Available in multiple languages, Omnitool creates world backups and maps worlds on launch. It comes with several plugins to provide more uses, and more plugins may be found on the TerrariaOnline forums.

Features:

  • Custom Made World Generators
    • Planetoids
    • Flatworld
    • Dungeon Arena
    • World from Image
  • World Mapping
    • Saves .png map files in your terraria saves directory, for viewing with image viewers.
    • Clicking on a world listing also renders the world similar to Terraria or Terrafirma.
  • Available in twelve languages.
  • Can render Worlds with 16x16 pixels per block, like Terraria
